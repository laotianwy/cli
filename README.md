### 使用命令创建项目模板
npx @tianzhitong/cli create 【project-name】

### 使用本包发布到npm。默认修改最右侧版本
npx @tianzhitong/cli publish

### 修改最左侧大版本
npx @tianzhitong/cli publish -l

### 修改中间版本
npx @tianzhitong/cli publish -m

### 修改右侧版本
npx @tianzhitong/cli publish -m
